# **MindCare AI: Mental Health Support Assistant**

## **Overview**

MindCare AI is a web based application that helps users understand their emotional well being. It analyzes user input text to detect mood and provides simple supportive responses. The system also allows users to track their daily mood and view patterns over time.

---

## **Features**

* Mood detection from text
* Daily mood tracking
* Basic AI chat support
* Suggestions based on mood
* Simple dashboard

---

## **Tech Stack**

* Frontend: HTML, CSS, JavaScript
* Backend: Flask
* Database: SQLite
